This project includes basic API testing using Python, Requests, and Pytest.

## Technologies used
- Python
- Requests
- Pytest

## Test scenarios
- Get a valid post
- Get an invalid post (404)
- Validate response content
- Create a new post (POST request)

## How to run

1. Install requirements:
pip install -r requirements.txt

2. Run the tests:
pytest test_api.py

## Notes
This project uses a public test API (JSONPlaceholder) for demonstration purposes.
